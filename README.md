# ComposeWeatherApp

Developed on Android Studio Chipmunk 2021.2.1 patch 1  
Tested on moto g play 2021 api 30  
It uses Jetpack Compose  
It has one sample unit test - WeatherUnitTest.testFromViewModelToRepositoryCall()

## Architecture

 - Compose UI
 - ViewModel
 - Repository
 - Api service
 - Room local database
 - DI with Hilt
